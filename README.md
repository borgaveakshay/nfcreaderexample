# NFC Card Reader

A modern Android application that reads NFC card information using Jetpack Compose and Material 3 design.

## Features

- Read NFC card details (VISA, Mastercard, AMEX, etc.)
- Beautiful Material 3 card display
- Support for ISO 14443-4 cards
- Real-time NFC status updates
- Error handling and user feedback
- Modern Kotlin & Compose implementation

## Tech Stack

- **Language**: Kotlin 2.0.21
- **Minimum SDK**: 24 (Android 7.0)
- **Target SDK**: 34 (Android 14)
- **Build System**: Gradle 8.7.3 with Kotlin DSL
- **Architecture**: MVVM

### Libraries & Dependencies

- **Jetpack Compose**: UI toolkit
- **Material 3**: Design system
- **Kotlin Coroutines**: Asynchronous programming
- **StateFlow**: Reactive state management
- **Android NFC**: NFC card reading capabilities
